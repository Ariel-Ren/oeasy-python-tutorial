###  存储位置

- i_age 在内存中的地址
	- 在 id(i_age)

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20221030-1667135565149)

- 第一句的赋值过程
  - 在内存中分配一块空间 存储 整数
	  - 分配空间的位置 在 id(i_age)
	  - 然后把分配空间地址给 i_age 变量
- i_age
	- 类型 class 是 int
	- 内存地址 是 id(i_age)
- 整型变量也可以给其他变量赋值吗？

### 赋值

```
i1 = 0
i2 = 1
print(id(i1),id(i2))
i2 = i1
print(id(i1),id(i2))
```
